A Gentle Introduction to Graph Neural Networks

Sep 23, 2022 in Paper Notes

There are many practical applications of graph neural networks in areas such as antibactertial discovery, physics simulation, fake news detection, traffic prediction and recommendation systems. This articles explore and explains modern GNN.

What a graph is

  • vertex attributes
  • edge attributes and directions
  • global attributes

to further describe each node, egde or the entire graph, we can store information in each of these pieces of the graph (using vector).

Graph and where to find them

image as graphs: each pixel represents a node and is connected via an edge to adjacent pixels. The information stores in each node is a 3-d vector.

text as graphs: each word and character is a node and is connected via an edge to the node that follow it (a simple directed graph).

In practice, this is not usually how text and images are encoded.

molecules as graphs:it's convenient to describe this 3D object as a graph, whose nodes are atoms and edges are covalent bonds.

social network as graphs: modeling individuals as nodes and their relationships as edges.

citation networks as graphs: each paper is a node and each directed edge is a citation between one paper and another.

other examples: objects in visual scenes; variables in ML models, progamming code and math equations.

What types of problems have graph structured data

graph-level task: predict the property of en entire graph.

  • molecule
  • image classification
  • sentiment analysis of text

node-level task: predict the identity or role of each node within a graph.

  • people's role in a social network
  • image segmentation
  • determine the part of speech of a word

edge-level task: predict the relationship of each node with edges

  • image scene understanding

The challenges of using graphs in machine learning

How we will represent graphs to be compatible with neural networks.

Graphs have up to four types of information: nodes, edges, global-context and connectivity. The first three can be represented with feature matrix N and the last one with adjacency lists.

Graph Neural Networks

A GNN is an optimizable transformation on all attributes of the graph (nodes, edges, global-context) that preserves graph symmetries (permutation invariances).

the simplest GNN

learning new embeddings for nodes, edges and global-context but not use connectivity.

  • use a MLP as a GNN layer
  • respectively, input node, edges and global-context vectors, get back learned nodes, edges and global-context embedding
  • stack GNN layers together

GNN predictions by pooling information

We need collect information from edges and give them to nodes for prediction by pooling. (on final layer)

  • gather each of their embeddings and concatenate
  • aggregated, usually via sum

edge to node / node to edge / node to global

Passing messages between parts of the graph

message passing, where neighboring nodes or edges exchange information, leverage the connectivity of graphs.

  • gather all the neighboring node embeddings (messages), g
  • aggragate all messages via an aggregate function (like sum)
  • pass through an update function, usually a learned neural network

Learning edge representations

We can share information between nodes and edges within the GNN layer using message passing by pooling. We could choose whether to update node embeddings before edge embeddings, or the other way around. The order we update them is one design decision.

Adding global representations

Nodes that are far away from each other in the graph may never be able to efficiently transfer information to one another. One solution: using a master node or context vector which connected to all other nodes and edges.

GNN playground

The GNN playground allows us to see how these different components and architectures contribute to a GNN's ability to learn a real task: a graph-level prediction task with small molecular graphs.
